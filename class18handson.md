Where are the mismatches? Where are the gaps?

``` r
mismatch.sites <- which(ide < 1)
gap.sites <- gap.inspect(seqs)$t.inds
```

Exclude the gaps

``` r
tumor.sites <- mismatch.sites[ !mismatch.sites %in% gap.sites ]
```

Now get sequences with all possible 9-mers

``` r
start.positions <- tumor.sites - 8
end.positions <- tumor.sites + 8
```
